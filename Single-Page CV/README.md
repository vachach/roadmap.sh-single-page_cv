# Single-Page CV

This project is a semantic single-page CV built as part of the [roadmap.sh Single-Page CV project](https://roadmap.sh/projects/single-page-cv).

## Features

- Semantic HTML structure
- Clean and simple single-page layout
- Contact information, skills, education, and experience sections
- SEO meta tags
- Open Graph meta tags
- Favicon support

## Technologies Used

- HTML5
